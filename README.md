## 🧱 VoxelForge

**VoxelForge** is a study project focused on recreating core systems of a voxel-based game (inspired by Minecraft) using **Godot + C#**.

The main goal is to implement procedural terrain generation and explore how voxel engines work under the hood.

---

## 🎯 Goals

- 🌍 Procedural terrain generation  
- 🧱 Chunk-based world system  
- ⚙️ Efficient voxel rendering  
- 🌱 Scalable architecture for future features  

---

## 🛠️ Technologies

- Godot Engine  
- C#  
- Voxel-based rendering techniques  

---

## 🚧 Current Status

This project is currently in development.

### ✔️ Implemented
- Basic voxel structure  
- Initial terrain generation experiments  

### 🔜 Planned
- Chunk system  
- Noise-based terrain (Perlin/Simplex)  
- Infinite world generation  
- Player controller

## 🧠 Learning Focus

- Data structures for 3D worlds
